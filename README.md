A responsive console-based Address Book application developed using C programming.
This project allows users to efficiently manage contacts with features like adding, searching, editing, deleting, and storing data using file handling.
Features :
1.Add new contacts
2.View all contacts (sorted alphabetically)
3.Search contacts (by name, mobile, email)
4.Edit contact details
5.Delete contacts
6.Save contacts to CSV file
7.Auto-load saved contacts on startup
8.Prevent duplicate mobile numbers and emails
9.Input validation for name, mobile, and email

Technologies Used :
C Programming
Structures
File Handling (CSV)
String Functions
